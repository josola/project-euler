Project Euler is a series of challenging mathematical/computer programming problems that will require more than just mathematical insights to solve. Although mathematics will help you arrive at elegant and efficient methods, the use of a computer and programming skills will be required to solve most problems.

- This repository has a [Style Guide](https://github.com/josola/project_euler/blob/main/docs/STYLE_GUIDE.md)! Be sure to check it out before making contributions.
 
- Speaking of contributions, this repository has a [Contributing Guide](https://github.com/josola/project_euler/blob/main/docs/CONTRIBUTING.md) as well.
 
- Each solution has its own Build Guide, be sure to check out their documentation within their respective directories!
